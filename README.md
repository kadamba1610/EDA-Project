# EDA Project

This is a basic Exploratory Data Analysis (EDA) project using Python.

- Python
- Pandas
- Matplotlib

- Data loading
- Data understanding (head, info, describe)
- Data analysis using groupby
- Data visualization using charts

## Key Insights
- Sales vary across cities
- Some products have higher quantity
- Data visualization helps understand patterns
